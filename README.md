![](https://github.com/Andyshome/AttendanceApp/blob/master/ScreenShot.PNG)



## How to build

* Clone the repository

```bash
git clone https://github.com/Andyshome/AttendanceApp.git
```

* Install the cocoapods in both projects

```bash
$ cd ClientSide
$ pod install
```

```bash
$ cd ServerSide
$ pod install
```

## Requirements
* Xcode 10.0
* IOS 12.0
* Swift 4.2
